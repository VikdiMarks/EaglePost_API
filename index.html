Об API

EAGLE POST API - это стандартный интерфейс логистических запросов. С помощью API вы можете запросить информацию о перевозки товарах, купленных вами в магазинах, с котороми мы сотрудничаем. Наше API основано на протокле HTTP и совместимо с различными языками разработки.

Использование API О запросе
  • Для отправки запросов используйте метод http POST.
  • Все запросы и ответы представлены в формате JSON и кодировке UTF-8.
  • Присутствует лишь один параметр для получения информации о перевозки посылки:
    • Номер отслеживания, который не должен быть комбинацией от 5 до 50 последовательных букв, цифр и/или дефиса.
  • Заголовок вопроса(обязательно):
    • Content-Type: application/json
  • Тело запроса (обязательно) должно быть в формате JSON

      <?php
      header('Content-Type: application/json');

      $curl = curl_init();

      curl_setopt_array($curl, [
        CURLOPT_URL => 'http://localhost/eagleApi/api/index.php/track',
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_FOLLOWLOCATION => true,
        CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
        CURLOPT_CUSTOMREQUEST => 'POST',
        CURLOPT_POSTFIELDS => '[
          {
            "number": "FC0124011513US",
            "carrier": 100003 
          }
        ]',
        CURLOPT_HTTPHEADER => [
          "17token: E7B0C9D6EA0A86D2B24F4339708E5234",
          'Content-Type: application/json'
        ],
      ]);
        
      $response = curl_exec($curl);
      $err = curl_error($curl);
        
      echo $response;              
     import requests

      url = "https://api.17track.net/track/v2/gettrackinfo"
      
      payload = [
          {
              "number": "270434455123",
              "carrier": 100003
          }
      ]
      headers = {
          "content-type": "application/json",
          "17token": "your secret key"
      }
      
      response = requests.request("POST", url, json=payload, headers=headers)
      
      print(response.text)              
     OkHttpClient client = new OkHttpClient();

      MediaType mediaType = MediaType.parse("application/json");
      String value = "[\r\n    {\r\n        \"number\": \"270434455123\",\r\n        \"carrier\":100003\r\n    }\r\n]";
      RequestBody body = RequestBody.create(mediaType, value);
      Request request = new Request.Builder()
          .url("https://api.17track.net/track/v2/gettrackinfo")
          .post(body)
          .addHeader("content-type", "application/json")
          .addHeader("17token", "your secret key")
          .build();
      
      Response response = client.newCall(request).execute();
      response.body();
